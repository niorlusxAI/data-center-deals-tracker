# 🚀 AI & Data Center / Cloud Spend Deals Tracker

Live tracker of hyperscaler + neocloud capex — Meta, Nvidia, AWS, CoreWeave, Equinix.
Turn deal signals into stock ideas for NVDA, AMD, VRT, EQIX.

## Deploy
1. Fork/clone this repo
2. Deploy on [Streamlit Cloud](https://streamlit.io/cloud)
3. Set secrets in Streamlit Cloud → Settings → Secrets:
   - `gmail_user`, `gmail_pass`
   - `stripe_checkout_url`
   - `adsense_code`, `ga_code`
4. Add paying emails to `premium_emails.csv`
